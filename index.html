Asteroid Belt

This was the project we worked on during our "Performant Rendering" class. A working game was provided, using an existing game engine, and our goal was to use modern optimisation techniques to improve frame times and memory utilisation.
For this, I implemented:

  • Entity Component System (ECS)
  • Removed redundant GPU bindings
  • Deferred rendering
  • Level of Detail (LOD) using MeshOptimizer
  • SIMD instructions
  • Broadphase collision detection
  • Instanced rendering
    • Throughout the project, I tracked and recorded performance differences using the Tracy profiler, assisted by RenderDoc and Nvidia Nsight usage.
